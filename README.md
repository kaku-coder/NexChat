# 🚀 NexChat

> **⚠️ Active Development**
>
> NexChat is an actively developed project. New features, performance improvements, and UI enhancements are continuously being added. The current version already demonstrates the core architecture of a scalable real-time messaging platform, with many exciting features planned for future releases.

---

## 📖 About

NexChat is a modern real-time messaging platform inspired by production-grade chat applications. The project focuses on building scalable backend architecture, secure authentication, and seamless real-time communication while following industry best practices.

This repository is continuously evolving as new features are implemented.

---

## ✨ Current Features

- 🔐 User Authentication (JWT)
- 🔒 Password Hashing (bcrypt)
- 💬 Real-Time Messaging
- ⚡ Socket.IO Integration
- 🗄️ MongoDB Database
- 🚀 Redis Token Blacklisting
- 🌐 REST API
- 📁 Clean MVC Architecture

## 🚀 Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/NexChat.git

cd backend
npm install
npm run dev

cd ../frontend
npm install
npm run dev
```
